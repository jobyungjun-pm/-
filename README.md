** 1. 목표**

- 실제 URL이 나오는 “배포 경험”을 제공
- 친구/지인에게 공유할 수 있는 수준까지

git a---

** 2. 내용**

1. [streamlit.io](<https://streamlit.io>) / Streamlit Cloud 접속, GitHub 연동
2. “New app” → 레포 선택 → `app.py` 지정 → 배포
3. 배포된 URL 접속해 실제 동작 확인
4. 작은 버그 수정 → 커밋 → Push → 자동 재배포 체험
5. 마지막에 학생 각자:
   - 내 서비스 URL
   - 스크린샷 2~3장
   - “내가 구현한 기능 한 줄 소개” 정리
